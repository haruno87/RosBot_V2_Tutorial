# 如何添加/修改文档

1):首先将仓库 clone 到本地

2):在终端运行 `pip install mkdocs`按照依赖

3):运行 `mkdocs serve`，然后在终端的网页链接中跳转，就可以到页面中，本地的修改保存后会自动同步文档

（注意需要启动 mkdocs serve 再保存才能修改成功，本地修改成功后发布到 github 仓库中）
